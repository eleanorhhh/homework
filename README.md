- 開始進入程式必須以Main進入
#### 上傳至git
- git clone + 網址
- git --help 列出所有指令
- git status 看狀態
- cd+ "資料夾位置" 跳到資料夾
- 右鍵stage change ; message 名稱 ; commit 
- git add .  增加資料夾內容
- git commit -m "名字" 
- git checkout -b name 取名
- git push 上傳至github
